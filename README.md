# Proshop-mern
###### 購物網站(react,express)
## 專案內容分為前端跟後端

![image](https://user-images.githubusercontent.com/44516782/115414614-c4424100-a228-11eb-9b6c-20b0e5f8f222.png)
###### 在frontend安裝npm，並啟動
```
npm i
npm start
```
###### 在最外層安裝npm與nodemon，並啟動
```
npm i
npm i nodemon
npm start
```
